# Migration Guide: v0.7 → v0.8

All changes are backward compatible unless explicitly noted.

---

## Summary

| Change                                                   | Type              | Breaking?         |
| -------------------------------------------------------- | ----------------- | ----------------- |
| `StatCardProps.HxSwap` typed `string` → `htmx.SwapStyle` | API tightening    | No (compile-time) |
| `TableHeader` + `TypedHeaders` for sortable columns      | New feature       | No                |
| `FormProps.Inline` for horizontal layouts                | New feature       | No                |
| `ButtonHTMLType` → `map[X]string` + `utils.Lookup`       | Internal refactor | No                |
| `feedbackIconName` / `lookupFeedbackStyle` removed       | Internal refactor | No                |
| `GridProps.Gap` typed enum (`GridGap`)                   | New feature       | No                |
| `CopyButtonProps.Href`                                   | New feature       | No                |
| `ImageProps.Rounded`                                     | New feature       | No                |
| `LoadMoreProps.InfiniteScroll`                           | New feature       | No                |
| `NotFound404Props.LinksTitle`                            | New feature       | No                |

---

## 1. `StatCardProps.HxSwap` → `htmx.SwapStyle`

**Before:**

```go
StatCard(StatCardProps{HxSwap: "innerHTML"})
```

**After:**

```go
StatCard(StatCardProps{HxSwap: htmx.SwapInnerHTML})
```

## 2. `TableHeader` + `TypedHeaders` (sortable columns)

```go
Table(TableProps{
    TypedHeaders: []TableHeader{
        {Label: "Name", Sortable: true, SortDirection: SortAsc, Href: "/users?sort=name&dir=desc"},
        {Label: "Role"},
    },
    Rows: rows,
})
```

## 3. `FormProps.Inline` (horizontal layout)

```go
Form(FormProps{Action: "/search", Method: FormGet, Inline: true}) {
    @forms.Select(forms.SelectProps{Name: "status", Label: "Status", Options: opts})
    @display.Button(display.ButtonProps{Text: "Filter", Type: display.ButtonHTMLSubmit})
}
```

## 4. `GridProps.Gap` typed enum

```go
Grid(GridProps{Cols: GridCols3, Gap: GridGapLG}) { /* children */ }
```

| Constant  | Class           |
| --------- | --------------- |
| GridGapSM | gap-2           |
| GridGapMD | gap-4 (default) |
| GridGapLG | gap-6           |
| GridGapXL | gap-8           |

## 5. `CopyButtonProps.Href`

When set, renders an `<a>` instead of `<button>`. Link still copies to clipboard on click.

## 6. `ImageProps.Rounded`

When true, adds `rounded-full` instead of `rounded-md`.

## 7. `LoadMoreProps.InfiniteScroll`

When true, adds `hx-trigger="revealed"` for auto-load on scroll.

## 8. `NotFound404Props.LinksTitle`

Configurable heading for the links section (default: "Popular pages").

---

## Internal changes (no consumer impact)

- 6 lookup maps converted to typed-key maps
- 14 new `IsValid()` methods (total: 30 across all enums)
- `errorpage.CauseItem.Code` changed from `string` to `Code` type
- `feedbackIconName` / `lookupFeedbackStyle` replaced with `utils.Lookup`
