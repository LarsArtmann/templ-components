# Migration Guide: v0.8.0 → v0.9.0

This guide covers all changes in v0.9.0 and how to adopt them.

## New Features

### `display.GridProps.Gap` — typed gap control

```go
// Before (v0.8.0): gap was hardcoded to gap-4
@display.Grid(display.GridProps{Cols: display.GridCols3})

// After (v0.9.0): control spacing via typed enum
@display.Grid(display.GridProps{Cols: display.GridCols3, Gap: display.GridGapLG})
```

| Constant              | CSS class |
| --------------------- | --------- |
| `GridGapSM`           | `gap-2`   |
| `GridGapMD` (default) | `gap-4`   |
| `GridGapLG`           | `gap-6`   |
| `GridGapXL`           | `gap-8`   |

### `display.CopyButtonProps.Href` — link variant

```go
// Before: only a <button> was rendered
@display.CopyButton(display.CopyButtonProps{Text: "Copy ID"})

// After: optionally render an <a> that also copies
@display.CopyButton(display.CopyButtonProps{Text: "Copy ID", Href: "/settings"})
```

### `display.ImageProps.Rounded` — circular images

```go
@display.Image(display.ImageProps{Src: user.Avatar, Rounded: true})
```

### `navigation.LoadMoreProps.InfiniteScroll` — auto-load on scroll

```go
@navigation.LoadMore(navigation.LoadMoreProps{
    Href:           "/items?cursor=abc",
    InfiniteScroll: true, // adds hx-trigger="revealed"
})
```

### `errorpage.NotFound404Props.LinksTitle` — configurable links heading

```go
@errorpage.NotFound404(errorpage.NotFound404Props{
    LinksTitle: "Quick links", // default: "Popular pages"
})
```

### `errorpage.WriteNotFound404` — convenience HTTP handler

```go
// In your HTTP handler:
errorpage.WriteNotFound404(w, r, errorpage.NotFound404Props{
    GoHomeHref: "/",
}, nonce)
```

## Internal Changes

### Deduplication sprint

6 sub-templates extracted across `errorpage`, `display`, and `feedback`:
`errorHeader`, `goBackScript`, `actionLinkBody`, `skeletonContainer`,
`definitionDetailContent`, and merged `overlayPanel` into `overlayShell`.

These are internal refactors — no consumer API changes. If you fork
internal templates, review the new `errorpage/shared.templ` and
`display/shared.go` files.

### Coverage boost

152 new test functions added. No behavioral changes.

## No Breaking Changes

v0.9.0 is fully backward compatible. All new features use opt-in fields
that default to existing behavior.
