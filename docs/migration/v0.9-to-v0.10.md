# Migration: v0.9 → v0.10

## Breaking Changes

### `errorpage.NotFound()` status code: 400 → 404

`NotFound()` now sets `StatusCode: 404` (was 400). If you relied on the old 400
status, set it explicitly:

```go
props := errorpage.NotFound()
props.StatusCode = 400
```

### `errorpage.Forbidden()` status code: 400 → 403

Same pattern. `Forbidden()` now returns 403.

### `errorpage.InternalError()` status code: 503 → 500

`InternalError()` now returns 500 (was 503).

## New Features

- **Dark mode compliance:** All components now have `dark:` variants. Class-based
  strategy via `layout.ThemeScript()` + `layout.ThemeToggle()`. Set `color-scheme:
light/dark` in your CSS for native form control rendering.
- **Motion-reduce:** All transitions/animations include `motion-reduce:*` fallbacks.
  Enforced by `utils.TestMotionReduceCompliance`.
- **RTL/i18n:** All CSS uses logical properties. Components auto-mirror under
  `dir="rtl"`.
- **`errorpage.StatusCode` field:** Override the family-derived status code on
  any `ErrorPageProps`.
- **`forms.RadioOption.Checked`:** Pre-select radio options for edit forms.
- **`forms.RadioProps.Required`:** Propagates `required` to individual radios.

## Bug Fixes (34 total)

See the [v0.10.0 CHANGELOG](../../CHANGELOG.md) for the full list of fixes.

Key fixes:

- `forms.Toggle` peer-checked classes now visible to Tailwind scanner
- `forms.Combobox` disabled hidden input, stale hidden value, Enter blocking
- `forms.Select` slice mutation corruption
- `display.Modal`/`Drawer` aria-hidden/inert sync on JS open/close
- `display.Dropdown` RTL arrow-key dead code
- `htmx.LoadingButton` default text now hides during loading
- `feedback.Toast` auto-dismiss now works with `DefaultToastProps()`

## CSS Setup

If upgrading from v0.9, add to your CSS entry point:

```css
@import "tailwindcss" source(none);
@source "**/*.templ";
@source "vendor/github.com/larsartmann/templ-components/**/*.templ";

@custom-variant dark (&:where(.dark, .dark *));
```

See `templates/app.css` for a complete starter.
