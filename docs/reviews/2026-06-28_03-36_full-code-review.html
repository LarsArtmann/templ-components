Honest verdict The library is well-built — green build, green tests, zero lint issues, no dupes/split-brains/bool-blindness. But the review surfaced real a11y bugs (modal/drawer focus + aria, spinner/progressbar announcements, combobox field linking), a security regression (MobileNavLink external), and correctness hazards (HTMX retry counter, empty-nonce CSP, errorpage 503 retry-storm). Full remediation plan in docs/planning/.
