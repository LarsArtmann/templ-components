## Safety Constraints

1. **Never break the build** — test after every change
2. **Never revert changes I didn't make** — adminui has another active agent
3. **adminui migration is hybrid first** — keep admin.css as `@layer base` until all pages are migrated, THEN delete it
4. **One .templ file at a time** — test after each conversion
5. **If adminui is too risky** — ship tc v0.4.0 and report back
